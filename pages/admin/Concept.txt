=> Pex Group Add

   - [Input] Gruppenname
   - [Input] Preffix
   - [Input] Suffix
   - [Input] Ranklevel
   - [Checkbox] Default Group
   

=> Pex Group Edit:

   -> Group Generell Settings:
      - [Input] Groupname
      - [Input] Preffix
      - [Input] Suffix
      - [Input] Rank level
      - [Checkbox] Default Group
	  
   -> Group Permissions:
      - Listing available Permissions for this Group
	  - [Input] Permissions
	  - [Input] World define


=> Pex User:

   - Detail Listing of User (Group, Playername, last Onlinetime) (Links: Edit Player)
   - Dropdownmenu for quick list Permissions
   
   
=> Pex User Edit:

   -> User Details:
      - [Input] Username
      - [Input] Uuid
      - [Select] Groupchange
	  - [Input] Prefix
	  - [Input] Suffix
	  
   -> User Permissions:
      - Listing available Permissions for this User
	  - [Input] Permissions
	  - [Input] World define
